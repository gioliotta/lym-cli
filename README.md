# lym: CLI Toolkit - en desarrollo
CLI diseñado para programadores v1.0.0

### Instalación
```bash
sudo apt -i lym-cli
```

### Uso
```sh-session
lym [command]

# Run `help` for detailed information about CLI commands
lym [command] --help
```
